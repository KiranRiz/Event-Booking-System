# EventHub — Event Booking Management System

EventHub is a full-stack web application built using the MERN stack that allows users to discover, book, and manage events online. The system supports three user roles: Admin, Organizer, and User, each with distinct access levels and functionalities.

## Tech Stack

### Frontend
- React.js: UI Framework
- Tailwind CSS v3: Styling & Responsive Design
- React Router DOM: Client-side Navigation
- Axios: HTTP API Calls
- React Hot Toast: Notifications
- Lucide React: Icons
- Context API: Global State Management

### Backend
- Node.js: Server Runtime
- Express.js: REST API Framework
- MongoDB Atlas: Cloud NoSQL Database
- Mongoose: Object Data Modelling (ODM)
- JWT (JSON Web Tokens): Authentication
- bcryptjs: Password Hashing

## User Roles
- User: Browse events, book tickets, manage bookings |
- Organizer: Create and manage own events, view bookings |
- Admin: Full system access, manage users, events, bookings |

## Architecture

This application follows the MVC (Model-View-Controller) design pattern.
